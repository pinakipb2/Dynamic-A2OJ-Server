## Development Setup

### Prerequisites

- Install [Node.js] which includes [Node Package Manager][npm]

### Setting Up a Project

Create workspace:

```
mkdir [PROJECT NAME]
cd [PROJECT NAME]
```

Install [Nodemon] globally and install all packages:

```
npm i package.json
```

Run the application:

```
npm dev
```
